# Support Policy

Pass Alarm is an open-source project. We welcome bug reports and contributions.
However, **free support is limited**.

## Supported versions
We support:

- The latest release on the `main` branch
- The latest public store release (official build)

We do not support:

- Old versions
- Unofficial builds or forks (please contact the fork maintainer)

## Before opening an issue
Please confirm:

- Notification permission is enabled
- Battery optimization settings are checked (Android)
- The device is not in a mode that may suppress sound/notifications (iOS Focus/Silent/DND)

## Bug reports (required template)
Bug reports must include:

- Device model
- OS version
- App version/commit hash
- Reproduction steps
- Expected vs actual behavior
- Logs (if available)

Issues missing required info may be closed.

## Feature requests
Feature requests are welcome, but acceptance is not guaranteed.

## Security issues
Do not open public issues for security vulnerabilities.
See SECURITY.md.
