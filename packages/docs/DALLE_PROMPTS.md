## Output Specifications

| Asset | Format | Sizes Needed |
|-------|--------|-------------|
| App Icon | PNG | 1024x1024 (source), Xcode/Android Studio will resize |
| Splash | PNG | 1170x2532 (iPhone), 1080x1920 (Android) |
| Illustrations | PNG (transparent) | 2x and 3x (@2x: stated size, @3x: 1.5x) |
| Tab Icons | SF Symbols preferred | If custom: 48x48 @1x, 96x96 @2x, 144x144 @3x |
| Notification Icons | PNG (transparent) | Android: 24x24, 48x48, 72x72, 96x96 mdpi-xxhdpi |
| Pro Assets | PNG (transparent) | 2x and 3x |
